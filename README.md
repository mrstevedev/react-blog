# Custom Blog

Custom Blog application built with React.js, Material UI, Express.js, & MongoDB.

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install dependencies.

## Inside /client

```bash
npm install
```

## Inside /

```bash
npm install
```
